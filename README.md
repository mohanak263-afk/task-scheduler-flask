# 📅 Task Scheduler

A simple and responsive **Task Scheduler Web Application** built using **Flask, HTML, CSS, and JavaScript**. The application helps users organize daily tasks by allowing them to add, edit, complete, and delete tasks through an intuitive web interface.

---

## 🚀 Features

- ➕ Add new tasks
- ✏️ Edit existing tasks
- ✅ Mark tasks as completed
- 🗑️ Delete tasks
- 📊 Dashboard displaying:
  - Total Tasks
  - Completed Tasks
  - Pending Tasks
- 🔍 Search tasks instantly
- 🎯 Filter tasks (All, Completed, Pending)
- 🏷️ Priority Levels (High, Medium, Low)
- 🌙 Dark Mode
- 📱 Responsive design for desktop and mobile

## 📖 How It Works

1. Enter task details.
2. Click **Add Task**.
3. View all tasks on the dashboard.
4. Mark tasks as completed.
5. Edit task details whenever required.
6. Delete completed or unwanted tasks.
7. Search and filter tasks for quick access.
8. Toggle Dark Mode for a better viewing experience.
